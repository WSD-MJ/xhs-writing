# Xiaohongshu (RED) Note Generator — Agent Skill

An AI Agent Skill that generates complete, publish-ready **Xiaohongshu (小红书 / RED / RedNote / XHS)** notes from a topic — title, body, emoji, and hashtags in authentic RED style.

Works with Claude Code, Cursor, Codex, OpenClaw and any agent that supports the open `SKILL.md` standard.

## What it does

Give it a topic (and optional style requirements) and it returns a fully-formatted note ready to post.

## Install / use

The skill lives in [`xiaohongshu-note-generator/SKILL.md`](./xiaohongshu-note-generator/SKILL.md).

1. Get an API key at https://ai.wsdsocial.com/skills
2. Set `WSD_API_KEY` as an environment variable
3. See the SKILL.md for the request format
